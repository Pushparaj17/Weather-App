# Weather-App
An Android App where you can check weather from city you want, it was built with mvvm, dagger-hilt, livedata,coroutines, jetpack libraries...

# Features

1- Kotlin  
2- MVVM design pattern  
3- Retrofit2   
4- ViewBinding    
5- Coroutine  
6- Weather Api  
7- LiveData  
8- Dagger-Hilt
9- JUnit
10- Espresso
